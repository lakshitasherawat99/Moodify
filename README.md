# Moodify
Moodify is a web-based application that converts user-input emotions into music. By typing a mood or feeling, the app maps it to a corresponding sound and visual experience.

## Features
- Mood-based music playback
- Interactive UI with visual feedback
- Animated interface with dynamic elements
- Sidebar for quick mood selection

- ## Tech Used
- HTML
- CSS
- JavaScript

## Why I built this
As someone interested in both science and music, I wanted to explore how systems can interpret emotional states and translate them into something tangible like sound.
